
# SBS 완료 플랜 정리

## 변수 설정

- `{PLAN}`: 첫 번째 인자

## 변수 검증 및 재설정

아래 규칙으로 `{PLAN}` 을 검증하고 재설정 해라.

- **`{PLAN}`이 비어있을 경우**:
    - `.sisyphus/plans/` 에서 `{PLAN}.md` 후보를 찾는다.
    - 후보가 0개면 아래를 출력하고 종료
        ```
        ❌ 삭제할 플랜을 찾지 못했습니다.
        RESULT: ERROR
        ```
    - 후보가 1개면 자동 선택
    - 후보가 2개 이상이면 `question` 도구로 선택

## 삭제 확인

삭제 전 반드시 `question` 도구로 아래 확인을 수행해라:

- header: "플랜 삭제 확인"
- question: `"{PLAN} 을 삭제하시겠습니까?"`
- options:
    - label: "삭제"
    - label: "유지"

응답이 "유지"면 아래를 출력하고 종료:

```
유지 선택으로 플랜 삭제를 취소했습니다.
```

## 삭제 실행

응답이 "삭제"면 아래 순서대로 처리해라:

1. `.sisyphus/plans/{PLAN}.md` 존재 시 삭제
2. 위 파일이 없으면 `.sisphus/plans/{PLAN}.md` 존재 시 삭제
3. 둘 다 없으면 아래를 출력하고 종료
    ```
    ❌ 삭제 대상 파일이 없습니다.
    ```

삭제 성공 시 아래를 출력하고 종료:

```
플랜 삭제 완료: {PLAN}
```
